🗃️ Orders 表新增字段

配送/自提模式引入后，订单表需要新增以下字段

字段 类型 说明
delivery_type 'pickup' | 'delivery' 配送/自提标识，默认 pickup
delivery_fee number 配送费金额（自提时为 0）
remark string? 通用订单备注（可选）
delivery_address string? 下单时的配送地址快照（配送时必填）

📌 注意

delivery_address 是下单时的地址快照，不是引用用户 profile 的地址。因为用户后续可能修改地址，但订单的配送地址应该固定为下单时的值。手机号同理，建议也存快照 delivery_phone。
